列表

  • 无序列表：在 Markdown 文件的文字前面添加“-”或者“*”号。
  • 有序列表：在 Markdown 文件的文字前面添加“1. 2. 3. ”这样的数字序号。
    符号或者数字，与正文之间使用空格分割。

效果

  • aaa
  • bbb
  • ccc
